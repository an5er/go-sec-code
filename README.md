# Go漏洞靶场
"漏洞类型"包括：
* 任意文件读取
* 任意文件写入
* SSRF
* 命令执行注入
* 身份认证
* 模板注入
* 并发攻击

# 目录说明
* research: 比较少见的漏洞和研究性质的安全问题
* unsafe: 常见的安全问题
* safe: 对比"unsafe"安全的编码方式
